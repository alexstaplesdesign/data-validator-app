Need help? Common validation errors explained

❌ Common Errors:

  • • Customer ID = 0 or negative
  • • Missing @ in email address
  • • Name too short (less than 2 chars)
  • • Future signup dates
  • • Invalid date format

✅ Valid Examples:

  • • customer_id: 1, 25, 1000
  • • email: user@domain.com
  • • full_name: Alice Johnson
  • • signup_date: 2025-01-30
